- 개발자 접근성 편의를 위해 `.geojson` 대신 카페 이름과 구글 맵 링크를 기재합니다.
- `개발자 카페`의 조건은 아래와 같습니다.
  - 카페에서 앉을 수 있는 좌석이 제공 될 것.

  - `4시간` 이상 이용이 가능할 것.
  - 와이파이가 제공 될 것.
